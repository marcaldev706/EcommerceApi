# MINI ECOMMERCE API

Projeto desenvolvido para estudo e prática de desenvolvimento backend utilizando Java e Spring Boot. O objetivo é aplicar conceitos aprendidos em documentações, artigos, livros e vídeos, com foco em boas práticas de desenvolvimento e construção de APIs REST.

---

## OBJETIVO DO PROJETO

Este projeto tem como finalidade praticar:

* Desenvolvimento de APIs REST
* Arquitetura em camadas
* Boas práticas de código
* Modelagem de entidades
* Persistência de dados com JPA/Hibernate
* Organização de projetos backend

O sistema simula um mini ecommerce contendo funcionalidades básicas de gerenciamento de usuários, produtos e pedidos.

---

## TECNOLOGIAS UTILIZADAS

* Java
* Spring Boot
* Spring Data JPA
* Hibernate
* Maven
* Banco de dados relacional
* Git e GitHub

---

## ESTRUTURA DO PROJETO

O projeto segue uma arquitetura em camadas:

* controller — camada responsável pelos endpoints da API
* service — regras de negócio da aplicação
* repository — comunicação com o banco de dados
* entity — entidades do sistema
* exception — tratamento de exceções
